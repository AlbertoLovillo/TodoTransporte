# TodoTransporte
Esta aplicacion esta hecha por Alberto García Lovillo, Dalton Eduardo Vargas Peralta y Borja Domínguez Castaño para el Proyecto Intermodular de Desarrollo de Aplicaciones Multiplataforma.

Consiste en un aplicación de transporte público de Málaga donde recoge autobuses, metro y tren de cercanías. Contiene varias herramientas para facilitar el uso de estos transportes, las cuales son:
- Mapa de rutas
- Mapa de puntos de venta
- Compra de billetes
- Creación de rutas personalizadas
- Lector de tarjetas
