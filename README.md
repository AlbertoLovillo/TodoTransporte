# TodoTransporte
Esta aplicacion esta hecha por Alberto García Lovillo, Dalton Eduardo Vargas Peralta y Borja Domínguez Castaño para el Proyecto Intermodular de Desarrollo de Aplicaciones Multiplataforma.

Consiste en un aplicación de la EMT de Málaga donde se pueden consultar información sobre las líneas y puntos de venta, además de comprar billetes. Las herramientas que presenta son:
- Mapa de Puntos de venta, ya sean estancos, kioskos o sedes oficiales de la EMT .
- Todos los horarios de cada parada segun la línea.
- Mapa de las líneas existenetes en la EMT con sus respectivas paradas y proxima hora.
- Cartera personal de billetes.
- Tienda de billetes para cada línea.
